RONIN MASK is a rhythm-based samurai action game that challenges your focus, reflexes, and sense of timing. Blending Rhythm Action, Reaction-based gameplay, and Quick Time Events (QTE), the game throws players into intense one-on-one sword duels where only absolute concentration can keep you alive. In RONIN MASK, you follow the journey of a lone Ronin who overcomes adversity through unwavering focus. Every battle is a test of perception, precision, and rhythm.

Game Controls:

A / S / D / J / K / L:  Rhythm Input / Action Timing
